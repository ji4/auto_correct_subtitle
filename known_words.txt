# known_words.txt - 常見需要校正的字詞
#
# 說明：
# 本檔案用於記錄常見需要校正的文字，每行一個校正規則
# 格式為：原文 > 修正後
# 
# 例如：
# 賬 > 帳
# 定價 > 訂價
#
# 注意事項：
# 1. 每行僅能包含一個校正規則
# 2. 原文和修正後的文字之間必須使用 ">" 符號分隔
# 3. ">" 符號前後可以有空格，會被自動忽略
# 4. 以 "#" 開頭的行會被視為註解，不會被處理
# 5. 替換是全局性的，將替換字幕檔中所有符合的文字
# 6. 替換預設不區分大小寫，例如 "iphone"、"IPHONE"、"iPhone" 都會被替換
# 7. 如果需要區分大小寫，請在行尾加上 [case]，例如：原文 > 修正後 [case]
#
